Folio

From raw research to finished manuscript.

♦

A Claude Code skill that takes your messy notes, data, and references and orchestrates them into a polished, export-ready paper — with human checkpoints at every critical turn.

Read the Docs View on GitHub
I.

Three acts, one command.

Folio handles the tedious mechanics of manuscript assembly so you can focus on the ideas.

01

Normalizes

PDFs, notes, datasets, BibTeX files — whatever you have. Folio classifies and structures your raw materials into a clean workspace.

02

Orchestrates

A six-stage pipeline with three human checkpoints. You approve direction before the skill commits to structure, prose, or export.

03

Exports

LaTeX, Markdown, or PDF. Folio compiles your manuscript with proper citations, figures, and formatting — ready for submission or review.

II.

Shaped to your paper.

Folio routes to the right mode based on your materials, or you can choose directly.

Mode I

White Paper

Narrative-driven, executive-facing. For positioning pieces, technical reports, and thought leadership that argues a thesis.

  • Markdown-first output
  • Narrative arc structure
  • Inline citations

Mode II

Research Paper

Evidence-heavy, LaTeX-forward. For empirical research with tables, figures, statistical results, and formal methodology.

  • LaTeX output with BibTeX
  • IMRaD structure
  • Figure and table management

Mode III

Hybrid

Combined framing and evidence. When your paper needs both a compelling narrative and rigorous empirical support.

  • Flexible output format
  • Adaptive structure
  • Mixed citation styles
III.

Eight commands. One pipeline.

Run the full pipeline with /folio, or invoke any stage independently.

Initialize

Create workspace, ingest materials

/folio:init

Prep & Route

Classify, validate, build ledger, select mode

/folio:prep

Plan

Outline structure and section plan

/folio:plan

Support

Gather evidence, build citations

/folio:support

Draft

Write sections in chosen mode

/folio:draft

Review

Redline check, IP scan, polish

/folio:review

Export

Compile and package deliverables

/folio:export

Automated stage
Human approval required
/folio:status — check workspace state anytime
IV.

Three lines to go.

Install the skill, run the full pipeline or a single stage.

Install & run.

# Install
git clone https://github.com/athan-dial/folio ~/.claude/skills/folio

# Run full pipeline
/folio "Your research idea" /path/to/materials

# Or run a single stage
/folio:status workspace/
